# Ondokai — Releases

Instaladores oficiais do app de desktop **Ondokai** (download manual).

O código-fonte é **privado**; apenas os **instaladores buildados** são publicados aqui como
**GitHub Releases**. Cada release `vX.Y.Z` traz:

| Plataforma | Arquivo |
|---|---|
| **macOS** (Apple Silicon) | `Ondokai-X.Y.Z.dmg` |
| **Windows** (x64) | `Ondokai-X.Y.Z-setup.exe` |
| **Linux** (x64) | `ondokai-X.Y.Z-x86_64.AppImage` |
| **Linux** (arm64) | `ondokai-X.Y.Z-arm64.AppImage` |

## Download

Pegue a versão mais recente em **[Releases → latest](../../releases/latest)**, ou pelos
links estáveis do site (redirecionam para o asset correto):

- **Windows** — https://ondokai-landing.web.app/download/windows
- **macOS** — https://ondokai-landing.web.app/download/macos
- **Linux** — https://ondokai-landing.web.app/download/linux

> ⚠️ Os instaladores ainda **não são assinados** (beta) — o macOS (Gatekeeper) e o Windows
> (SmartScreen) podem exibir um aviso de "desenvolvedor não identificado". A atualização é
> **manual**: baixe a versão nova quando o app avisar que há uma atualização.
